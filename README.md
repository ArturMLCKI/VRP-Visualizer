Demonstration project - im currently working in company that specialize in VRP (walking salesman) but im not working there as a programmer but this this is my goal so based on my expirence i want to create projects that is realted to this. This project is nothing special at all but I want to show you what i can do.

---

## Features

- Upload route data from CSV/TXT/XLSX files
- Manual route input
- Backend data processing
- Store processed routes in database
- Retrieve previous routes
- Delete saved routes
- Display routes on an interactive map
